# Results

**English (US)** | [Português (Brasil)](README.pt-BR.md)

`figures/` contains the selected final figures from the working copy. The clean
notebooks can regenerate the corresponding curves.

The `notebooks/` directory, created during execution, receives executed copies
of the notebooks and is ignored by Git to avoid versioning large embedded
outputs.
