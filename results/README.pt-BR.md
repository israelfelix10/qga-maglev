# Resultados

[English (US)](README.md) | **Português (Brasil)**

`figures/` contém as figuras finais selecionadas da cópia de trabalho. Os
notebooks limpos podem regenerar as curvas correspondentes.

O diretório `notebooks/`, criado durante a execução, recebe cópias executadas
dos notebooks e é ignorado pelo Git para evitar versionar grandes saídas
incorporadas.

arquivados e do código de análise. Elas são artefatos de pesquisa, não software,
